# Minimal's Public Package Registry Build Declarations

Build instructions for a curated, regularly updated set of open source packages
defined as build declarations using the
[Minimal Standard Types](https://github.com/gominimal/std) format.

## Current Status

This project is evolving quickly - more to come in the near future!

Thank you for your interest.
